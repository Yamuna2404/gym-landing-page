# 🏋️ Gym Landing Page

A modern **Gym / Fitness Landing Page** built to showcase plans, features, and comparisons in a clean and engaging UI.

---

## 🌟 Features

- ⚡ Modern Hero Section with strong CTA
- 💳 Pricing Section with 3 plans (Free / Pro / Premium)
- 🔄 Compare Plans Table for better decision-making
- 🎯 Clean UI with hover effects & smooth layout
- 🧩 Simple and structured footer

---

## 🖼️ Sections Overview

### 🔹 Hero Section
- Eye-catching headline
- Call-to-action button (Join Now / Get Started)
- Clean layout to grab user attention instantly

---

### 🔹 Pricing Section
- Three pricing cards:
  - Free Plan
  - Pro Plan
  - Premium Plan
- Hover effects for better interactivity
- Highlighted recommended plan

**Example Features:**
- 👤 Users Allowed
- 💾 Storage Access
- 🏋️ Workout Plans
- 📞 Support Level

---

### 🔹 Compare Plans
- Tabular comparison of all plans
- Helps users clearly see differences
- Improves conversion by simplifying decisions
